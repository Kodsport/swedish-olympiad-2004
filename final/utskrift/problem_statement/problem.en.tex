\problemname{Printout}

Pelle has a text entered into his computer that he now wants to edit and print on the printer. He wants the printout in a ``monospaced'' font. That is, all characters have the same width (take up the same amount of space on a line). Furthermore, he wants to adapt the printout to a given line length of $n$ characters per line.

He realizes that it is hardly possible to have all lines contain exactly $n$ characters, since there must be exactly \textbf{one} space between two words. However, he is satisfied if he can format the printout so that he gets as close to the desired result as possible. As a measure of the result, he defines the deviation in number of characters of the ``worst'' line from the desired $n$ characters. He now wants to minimize this measure.

Write a program that reads the text and prints it with line breaks at such positions that the above measure is minimized.

\section*{Input}
The input starts with a line specifying $5 \le n \le 80$. On the next line is a number $1 \le k \le 1000$ specifying the number of words, none longer than 20 characters. Then follow $k$ lines with one word on each line. The words contain no spaces, but may contain punctuation which is treated as part of the word and must not be separated from it. The text to be printed consists of these $k$ words in this exact order. The words contain punctuation and the letters $a$--$z$ and $A$--$Z$.

\section*{Output}
The program should print the text with line breaks.
